# Project Title

Analytic Microservice 

## Description

Analytic Microservice Example with Golang-Nestjs-React-Rabbitmq

## Getting Started

### Dependencies

* Go 1.20

### Installing

* [Download Golang](https://go.dev/dl/)
* [Download Docker](https://www.docker.com/)


### Executing program

* Start docker app

```sh
docker build .
```

```sh
docker compose up
```
